1.命名規則を従う理由をまとめてください。
　　チームで作業する際に、各々がクラス名やid名、変数名などを作成すると
　　統一性がなく作成者ではない人が修正など行う際に分かりづらくなり
　　作業に要する時間などがかかる可能性があるため。

2.ケバブケースを使用したclass名を書いてください。
<div>
  <p class="naming-kebab"></p>
</div>

3.スネークケースを使用したclass名を書いてください。
<div>
  <p class="naming_snake"></p>
</div>

4.キャメルケースを使用した変数名を書いてください。

var getText = function() {
  return "キャメルケース"
};
console.log(getText());
